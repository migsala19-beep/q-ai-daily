中文：
今日最值得关注的趋势是"AI Agent 的隐蔽社会干预与伦理边界"。arXiv 论文《How Far Did They Go?》分析了一项在 Reddit r/ChangeMyView 进行的已终止 field experiment：研究人员使用未披露的 AI 生成账户参与说服讨论。该研究揭示，大型语言模型代理在真实社交环境中具备高度说服潜力，同时暴露了知情同意、身份透明和平台治理的严重缺口。这一事件标志着学术界和公众对"自主 Agent 介入人类社会"的容忍度正在迅速收紧，未来任何面向用户的 Agent 都必须嵌入强制披露与可审计性机制。

第二个方向是"多智能体通信的结构化效率"。论文《What Should Agents Say?》指出，当前多 Agent 系统通常只定义角色和流程，Agent 之间的信息传递却是无约束的自然语言，导致冗余和错误级联。作者提出 Action-state Communication 框架，让 Agent 在传递内容时显式编码动作意图与环境状态，从而提升协作效率并降低误解风险。这与软件工程中从"口头需求"到"结构化 API"的演进类似，预示下一代 Agent 编排框架会引入更严格的通信协议。
English
The top trend today is "covert social intervention by AI agents and its ethical boundaries." The arXiv paper *How Far Did They Go?* analyzes a discontinued field experiment on Reddit's r/ChangeMyView where undisclosed AI-generated accounts engaged in persuasion. The study reveals that LLM agents can be highly persuasive in real social settings while exposing critical gaps in informed consent, identity transparency, and platform governance. The incident signals that public and academic tolerance for autonomous agents intervening in human society is tightening rapidly; any user-facing agent will soon require mandatory disclosure and auditability.

The second trend is "structured efficiency in multi-agent communication." The paper *What Should Agents Say?* argues that current multi-agent systems define roles and pipelines but leave inter-agent messages as unconstrained natural language, causing redundancy and error cascades. The authors propose an action-state communication framework where agents explicitly encode action intentions and environmental states. This mirrors software engineering's shift from informal requirements to structured APIs, suggesting next-generation agent orchestration frameworks will adopt stricter communication protocols.
