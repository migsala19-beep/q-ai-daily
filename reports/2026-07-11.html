🔬 深度研究报告

今日最值得关注的趋势是 AI Agent 的"过程化评估"与"推理能力升级"两条主线。
其一，AgentLens 提出对编码 Agent 应评估整条交互轨迹，而非仅看最终是否通过。这反映了业界痛点：生产环境中用户实际体验的是 Agent 逐步推理、纠错的完整过程，单一结果指标无法捕捉幻觉、循环、低质量交互等问题。
其二，"When Does In-Context Search Help?" 从采样复杂度理论切入，研究 LLM 如何通过迭代生成、批判、修正来完成推理。这意味着未来模型不必一味依赖更大规模训练，而可通过更好的上下文搜索策略提升复杂任务表现。两者结合，Agent 不仅要做对，还要"想得清楚、过程可解释"。
English
Two key trends stand out today: process-oriented AI agent evaluation and advanced reasoning capabilities.
AgentLens argues that coding agents should be assessed across the entire interaction trajectory, not just final pass/fail outcomes. This addresses a real production gap: users experience how agents reason, recover, and communicate, which single-bit metrics miss.
Meanwhile, "When Does In-Context Search Help?" offers a theoretical framework for reflection-driven reasoning, showing how iterative generation, critique, and revision can improve performance without simply scaling training. Together, these directions push agents toward both better outcomes and more transparent, interpretable reasoning.
